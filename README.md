## Setup

1. Clone this repository
2. Open `index.html` in a browser
3. For GitHub Pages deployment, push to the `main` branch

## Deployment

This repository is configured for GitHub Pages. Simply push to the main branch and GitHub will automatically deploy the site at:

`https://manikantatalluriaiengineer-spec.github.io/manikantatalluri.github.io/`

## Customization

- Update project data in `script.js`
- Modify styles in the `<style>` section of `index.html`
- Add resume PDF file as `resume.pdf` for download functionality
